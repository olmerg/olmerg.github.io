Support Vector Machine (SVM) - Tablero Interactivo

Explora la optimización del hiperplano y la geometría de los vectores de soporte.

Configuración

C = 1.0

Controla el compromiso entre margen y error.

Métricas de Optimización

Vectores de Soporte: 0
Ancho del Margen (\(2/||w||\)): 0.00
Nota: Los vectores de soporte son puntos \(x_i\) donde \(|D(x_i)| \approx 1\).
Haz clic en el lienzo para agregar puntos de diferentes colores.

TEORÍA ¿Qué está calculando trainSVM?

1. Función de Decisión

\( D(x) = \langle w, x \rangle + b \)

Calculamos el producto punto de los pesos \(w\) por tus coordenadas \(x\).

2. La Condición del Margen

\( y_i (\langle w, x_i \rangle + b) \ge 1 \)

Si esta condición es < 1, el punto es un Vector de Soporte o un error.

3. Optimización en código

// Derivada de Regularización
grad_w = w;
// Derivada de Error (Hinge)
if (margin < 1) {
  grad_w -= C * y * x;
  grad_b -= C * y;
}

El descenso de gradiente minimiza la norma de \(w\) (máximo margen) penalizando puntos intrusos según \(C\).
